# Examples

Small, self-contained, open engineering examples.

- [cantilever-beam](cantilever-beam/) — tip deflection of a cantilever beam
  under a point load at the free end (formula, worked numbers, Python script).
- [freedom-ship](freedom-ship/) — concept-level weight / area / hydrostatics
  model for a very-large floating city (design basis, generator script, data).

Contributions welcome — please open a pull request.
